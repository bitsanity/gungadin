# Public UI

User Interface available to anyone/everyone to identify, retrieve and decrypt
own files.

## Dependencies
1. Java Runtime Environment with updated security policy file
2. bitsanity/cryptils
  * https://github.com/bitsanity/cryptils
3. Lambdaworks SCrypt library
  * Apache 2.0
  * https://github.com/wg/scrypt
4. Optimized C library for EC operations on curve secp256k1
  * MIT License
  * https://bitcoin-core/secp256k1
5. LGoodDatePicker
  * java swing library for date selector widget
  * MIT License
  * https://github.com/LGoodDatePicker/LGoodDatePicker
6. Orchid TOR Client and Library (maybe)
  * TODO - support/documentation/examples lacking
  * https://subgraph.com/orchid/index.en.html
